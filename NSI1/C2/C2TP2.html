    TP 2 ASCII Art

    Objectifs

    • Utiliser des techniques de programmation de base impérative : boucles, tests, création et appels de fonction etc.
    • Savoir utiliser les spécicifications d'une fonction;
    • Manipuler le type chaîne de caractères en python (objectif secondaire).

    À savoir

    Ce TP comporte des zones de code à compléter, non évaluées et auto-corrigées. Il est cependant essentiel de traiter ces parties honnêtement. Le contenu est fortement inspiré d'un TP du livre NSI Hachette 1re.

    Introduction⚓︎

    L'ASCII Art consiste à utiliser les caractères alphanumériques du code ASCII pour produire des images. Voir la page wikipedia pour une description et quelques exemples.

    On commencera très simplement par dessiner un carré à l'aide de la lettre X. Puis, on augmentera progressivement la difficulté.

    Préambule: concaténation et répétition des chaînes de caractères⚓︎

    Python possède des types d'objets numériques natifs:

    • le type int (integer) pour manipuler les entiers relatifs;
    • le type float pour manipuler les nombres décimaux (dit à virgule flottante).
    • Pour manipuler les chaînes de caractères, on dispose du type str (string). Ces dernières sont déclarées en étant entourées de double quotes " " ou de simples quotes ' ' (s'il n'y a pas d'ambiguités avec l'apostrophe). Par exemple: 

    1
    mavariable = "programmation"
    La fonction print permet d'afficher une chaîne de caractères en terminant par un retour-chariot (ou retour à la ligne). Durant ce TP, on calculera d'abord le contenu d'une chaîne de caractères (souvent dans une fonction) avant de l'afficher. Pour cela, on peut utiliser des boucles :

    Warning

    Pour inclure une barre oblique inversée ('\' ou « backslash ») dans une chaîne de caractères, pour le bord droit du sapin, il faut répéter celle-ci : '\\'. La raison est la suivante : certains caractères spéciaux sont représentés dans les chaînes de caractères Python par un backslash suivit d’un code, par exemple \n pour le retour chariot et \t pour la tabulation.
