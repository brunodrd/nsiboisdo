    TP - Espionner les communications entre Bob et Alice⚓︎

    Bob et Alice communiquent en utilisant un chiffrement construit en 1974, mais qui n'a été publié qu'en 1978. Il permet à deux parties de se mettre d'accord sur un secret en commun par l'échange de messages, et sans que ces partis n'aient préalablement de secret commun.

    La technique⚓︎

    Supposons qu'Alice et Bob veulent communiquer en utilisant une clé secrète commune. Pour établir cette clé, Bob envoie à Alice un message qui consiste en un très grand nombre de puzzles, chacun assez petit pour qu'Alice puisse le résoudre par attaque par force brute. Les puzzles sont de la forme d'un message chiffré avec une clé inconnue, donc la clé est assez courte.

    Bob envoie tous les messages à Alice. Elle en sélectionne un au hasard et le résout par force brute. Le message chiffré contient un identificateur (du choix d'Alice) et une clé secrète, dite clé de session. Alice renvoie l'identificateur à Bob, qui sait alors également quelle est la clé de session.

    Exemple⚓︎

    Voici un exemple de message (pas encore chiffré) pouvant être généré par Bob :

    identiﬁcateur : 135674 – clé : fQ98c~m{)*e9+G!-
    identiﬁcateur : 426891 – clé : VMQ>Dm2|uyFU8$z\\
    identiﬁcateur : 734562 – clé : H\\>hq]D/x.K\\-Ov}
    identiﬁcateur : 924252 – clé : nSV)ayafjvzA9zXz

    Ce message est constitué d'un indentificateur numérique et d'une clé composée de 16 caractères.

    Ce fichier sera chiffré en utilisant le chiffrement XOR, avec une clé différente pour chaque ligne.

    Dans notre exemple ...

    La clé devra être assez courte pour que le déchiffrement d'une ligne ne soit pas trop long.

    Alice choisira une ligne au hasard dans ce ﬁchier. Ensuite, elle déchiffrera cette ligne en testant toutes les clés possibles. Admettons qu’elle choisisse et déchiffre la ligne suivante :

    identiﬁcateur : 924252– clé : nSV)ayafjvzA9zXz

    Alice transmettra en clair 924252 sur le réseau à destination de Bob.
    Il saura alors que la clé de session à utiliser sera nSV)ayafjvzA9zXz.
    Alice et Bob communiquerons alors avec cette clé lors de leurs prochains échanges.

    Le pirate Ève, a la tâche plus difficile. Il ne sait pas quel puzzle a été résolu. La meilleure stratégie pour Ève est de résoudre tous les puzzles, ce qui est beaucoup plus coûteux pour lui que pour Alice.

# Fonctions de chiffrement de type XOR
# La fonction c(message,cle) permet de chiffrer un message avec une clé en utilisant le chiffrement XOR. 
# Le retour sera une chaîne de caractère constituée des codes UTF-8 des caractères chiffrés.

# La fonction d(message,cle) permet de déchiffrer un message avec une clé en utilisant le chiffrement XOR. 
# En entrée, message est une chaîne de caractère constituée des codes UTF-8 des caractères chiffrés.
# Le retour sera une chaîne de caractère constituée des codes UTF-8 des caractères chiffrés.


def texte_2_UTF(texte : str)->list:
    tab_codes_UTF = []
    for c in texte:
        tab_codes_UTF.append(ord(c))
    return tab_codes_UTF

assert texte_2_UTF("MESSAGE") == [77, 69, 83, 83, 65, 71, 69]

# Python peut effectuer l'opération XOR directement avec les entiers, il n'est 
# donc pas nescessaire de convertir en binaire le message
# L'opérateur XOR s'effectue avec ^ en python

def c(message,cle):
    # Conversion des textes en codes UTF-8
    message_clair = texte_2_UTF(message)    
    tab_cle_initial = texte_2_UTF(cle)
    tab_cle = []
    n,m = len(message_clair),len(tab_cle_initial)
    # Répète la clé autant de fois que nescessaire
    for i in range(n):
        tab_cle.append(tab_cle_initial[i%m])
    message_chiffre = [] 
    # Effectue le XOR entre le message et la clé.   
    for i in range(n):
        message_chiffre.append(message_clair[i] ^ tab_cle[i])
    texte_chiffre = ''
    for c in message_chiffre:
        texte_chiffre += str(c)+' '
    return texte_chiffre


def d(message_chiffre,cle):
    # Conversion des textes en codes UTF-8
    message_chiffre = message_chiffre.split()
    tab_cle_initial = texte_2_UTF(cle)
    tab_cle = []
    n,m = len(message_chiffre),len(tab_cle_initial)
    # Répète la clé autant de fois que nescessaire
    for i in range(n):
        tab_cle.append(tab_cle_initial[i%m])
    message_clair = [] 
    # Effectue le XOR entre le message et la clé.   
    for i in range(n):
        message_clair.append(int(message_chiffre[i]) ^ tab_cle[i])
    texte_dechiffre = ''
    for c in message_clair:
        texte_dechiffre += chr(c)
    return texte_dechiffre
